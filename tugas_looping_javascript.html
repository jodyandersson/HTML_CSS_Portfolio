Macam-Macam Looping

Pada Javascript

Berikut ini adalah macam-macam looping pada javascript seperti for loop, for/in loop, for/of loop, while loop, dan do/while loop.


For Loop

Contoh Sintaks

<p id="ForLoop"></p> <script> let text = ""; for (let i = 0; i < 5; i++) { text += "Angkanya adalah " + i + "<br>"; } document.getElementById("ForLoop").innerHTML = text; </script>

Hasil For Loop di Javascript

For/In Loop

Contoh Sintaks

<p id="ForInLoop"></p> <script> const numbers = [45, 4, 9, 16, 25]; let txt = ""; for (let x in numbers) { txt += numbers[x] + "<br>"; } document.getElementById("ForInLoop").innerHTML = txt; </script>

For/Of Loop

Contoh Sintaks

<p id="ForOfLoop"></p> <script> const cars = ["BMW", "Volvo", "Mini"]; let mobile = ""; for (let x of cars) { mobile += x + "<br>"; } document.getElementById("ForOfLoop").innerHTML = mobile; </script>

While Loop

Contoh Sintaks

<p id="WhileLoop"></p> <script> let number = ""; let i = 0; while (i < 10) { number += "<br>Pengulangan ke-" + i; i++; } document.getElementById("WhileLoop").innerHTML = number; </script>

Hasil While Loop di Javascript

Do/While Loop

Contoh Sintaks

<p id="DoWhileLoop"></p> <script> let numb = "" let z = 0; do { numb += "<br>Ini Perulangan ke-" + z; z++; } while (z < 10); document.getElementById("DoWhileLoop").innerHTML = numb; </script>
