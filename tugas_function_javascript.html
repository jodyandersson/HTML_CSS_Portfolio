Macam-Macam Function

Pada Javascript menggunakan parameter

Berikut ini adalah macam-macam function di javascript menggunakan parameter seperti Default Parameter, dan Rest Parameter, dan Argument Object.


Default Parameter

Contoh Sintaks

Contoh 1
<p id="default_parameter"></p> <script> function myFunction(x, y) { if (y === undefined) { y = 2; } return x * y; } document.getElementById("default_parameter").innerHTML = myFunction(8); </script>
Contoh 2
<p>jika nilai b tidak diketahui, maka b = 10:</p> <p id="demo"></p> <script> function myFunction(a, b = 10) { return a + b; } document.getElementById("demo").innerHTML = myFunction(5); </script>

Rest Parameter

Contoh Sintaks

<p>Rest Parameter memungkinkan sebuah fungsi untuk memperlakukan jumlah argumen yang tidak terbatas sebagai sebuah array:</p> <p id="rest_param"></p> <script> function sum(...args) { let sum = 0; for (let arg of args) sum += arg; return sum; } let c = sum(4, 9, 16, 25, 29, 100, 66, 77); document.getElementById("rest_param").innerHTML = c; </script>

Argument Object

Contoh Sintaks

Contoh 1
<p>Mencari nilai terbesar.</p> <p id="demo2"></p> <script> function findMax() { let max = -Infinity; for(let i = 0; i < arguments.length; i++) { if (arguments[i] > max) { max = arguments[i]; } } return max; } document.getElementById("demo2").innerHTML = findMax(4, 5, 6); </script>
Contoh 2
<p>Jumlah dari seluruh Argumen:</p> <p id="demo3"></p> <script> function sumAll() { let sum = 0; for(let d = 0; d < arguments.length; d++) { sum += arguments[d]; } return sum; } document.getElementById("demo3").innerHTML = sumAll(1, 123, 500, 115, 44, 88); </script>
